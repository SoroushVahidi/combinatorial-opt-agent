# Results Provenance

**Status:** Authoritative — canonical evidence source for the EAAI manuscript  
**Last updated:** 2026-04-03

This document records the canonical artifact sources, benchmark denominators, and
provenance chain for all results cited in the EAAI manuscript.  It is the single
authoritative reference for questions about "which number came from where."

---

## Canonical benchmark setup

| Parameter | Value |
|-----------|-------|
| **Benchmark dataset** | NLP4LP (`udell-lab/NLP4LP`, gated HuggingFace) |
| **Query variant (primary)** | `orig` — original natural-language queries |
| **Query variants (robustness)** | `noisy` (numeric placeholders), `short` (first sentence only) |
| **Benchmark denominator** | **331 test queries** |
| **Catalog size (manuscript)** | **331 entries** (NLP4LP test schemas only) |
| **Catalog size (expanded)** | 335 entries (4 additional public-source entries added) |
| **Primary retrieval baseline** | TF-IDF cosine similarity |
| **Dense baselines** | SBERT, E5, BGE — supplementary only, not in main results table |

### 331 vs 335 catalog note

The EAAI manuscript reports results using the **331-entry NLP4LP catalog**.
The expanded 335-entry catalog was built post-manuscript by adding 4 entries from
public sources (NL4Opt, OR-Library).  Retrieval metrics with the 335-entry catalog
differ by at most ±0.003 R@1; downstream metrics are unaffected.  
Use the 331-entry catalog for any manuscript-consistent comparison.

### Optional LLM baseline reruns (not camera-ready Tables 1–5)

Two-stage LLM baselines (`--method openai` / `gemini` / `mistral`) write **auxiliary** CSV/JSON under a chosen output directory (historically `results/paper/` for some OpenAI runs; newer reruns use **`results/rerun/<provider>/run_<SLURM_JOB_ID>/paper/`**).

| Provider | Provenance doc | Typical committed evidence (if a run completes) |
|----------|----------------|--------------------------------------------------|
| OpenAI | (historical) | `results/paper/nlp4lp_downstream_*_openai.*` (partial set; not headline Table 1) |
| Gemini | [`docs/GEMINI_RERUN_REPORT.md`](GEMINI_RERUN_REPORT.md) | `results/rerun/gemini/run_*/preflight_*.json`, per-query CSVs, optional interrupt JSON |
| Mistral | [`docs/MISTRAL_RERUN_REPORT.md`](MISTRAL_RERUN_REPORT.md) | `results/rerun/mistral/run_*/preflight_*.json`, `mistral_sbatch_meta_*.json`, per-query `*_mistral*.csv`, optional `*.interrupt.json` |

Treat any LLM CSV as **non-interchangeable** with Table 1 without reading column definitions and the rerun report. **Absence** of `results/rerun/mistral/…` (or incomplete row counts) means **no full Mistral benchmark claim**.

## Blockers for full reproduction

| Blocker | Impact |
|---------|--------|
| `udell-lab/NLP4LP` gated on HuggingFace | Downstream metrics (TypeMatch, Exact20, InstReady) require `HF_TOKEN` |
| GPU / `torch` required | Learned retrieval fine-tuning cannot run CPU-only |
| No external LP/ILP solver | Solver-backed results use SciPy HiGHS shim (restricted to 20-instance subset) |

For workarounds, see `HOW_TO_REPRODUCE.md`.
