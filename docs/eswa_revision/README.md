# docs/eswa_revision — ESWA-era revision materials

This folder contains documentation and notes from the ESWA (Expert Systems With Applications)
revision phase, which preceded the current EAAI submission.

**These files are NOT authoritative for the current EAAI manuscript.**

They are retained for research provenance only. For the current canonical framing, use:

- [`docs/EAAI_SOURCE_OF_TRUTH.md`](../EAAI_SOURCE_OF_TRUTH.md) — manuscript authority
- [`docs/CURRENT_STATUS.md`](../CURRENT_STATUS.md) — reviewer-facing status summary
- [`docs/RESULTS_PROVENANCE.md`](../RESULTS_PROVENANCE.md) — canonical metrics and provenance chain
- `results/paper/eaai_camera_ready_tables/` — camera-ready tables (Tables 1–5)

**Note:** The ESWA-era experiment result tables in
`results/eswa_revision/13_tables/deterministic_method_comparison_orig.csv`
**are** the provenance source for the Table 1 canonical metrics (see
`docs/RESULTS_PROVENANCE.md`), even though the broader ESWA revision framing
is superseded.
