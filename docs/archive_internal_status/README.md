# Internal status & decision archives (provenance only)

This folder holds **non-canonical** narrative reports: manuscript-vs-repo comparisons, internal
go/no-go notes, literature-informed reruns, and similar **planning / audit** artifacts.

They are **retained for provenance** (who ran what, when, and under which constraints) and for
historical context during the EAAI submission path.

**They are not the public summary of results.** For reviewer-facing, manuscript-aligned status,
use:

- **[`docs/CURRENT_STATUS.md`](../CURRENT_STATUS.md)** — single current status page  
- **[`docs/EAAI_SOURCE_OF_TRUTH.md`](../EAAI_SOURCE_OF_TRUTH.md)** — manuscript authority + file list  
- **`results/paper/eaai_camera_ready_tables/`** — camera-ready tables (Table 1–5)

If you cite numbers for the paper or a public README, take them from the camera-ready tables and
`EAAI_SOURCE_OF_TRUTH.md`, not from the files here, unless you explicitly need a dated audit trail.
