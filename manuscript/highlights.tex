\noindent\textbf{Highlights}
\begin{itemize}
    \item We study retrieval-assisted instantiation for natural-language optimization.
    \item The pipeline retrieves a schema and deterministically fills scalar slots.
    \item TF--IDF, BM25, and LSA achieve strong fixed-catalog schema retrieval.
    \item The main remaining bottleneck is downstream number-to-slot grounding.
    \item Restricted subset studies support downstream engineering relevance.
\end{itemize}
